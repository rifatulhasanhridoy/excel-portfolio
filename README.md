# Excel Analytics Masterclass Portfolio

Welcome to my comprehensive Excel analytics portfolio! This repository serves as a centralized, structured database showcasing my progression through data cleaning, entry governance, and advanced calculation design.

---

## 🗂️ Portfolio Modules

### 📁 [01_Data_Cleaning](./01_Data_Cleaning/)
**Focus: Structural Data Preprocessing & Pipeline Building**
* **Objective:** Transforming chaotic, multi-source tracking records into pristine, analysis-ready tabular models.
* **Core Techniques Used:** Strip hidden structural spacing errors (`TRIM`), fix inconsistent text casings (`PROPER`, `UPPER`), isolate fragmented data blocks (`Text to Columns`), and eliminate exact record redundancy.

### 📁 [02_Data_Validation](./02_Data_Validation/)
**Focus: Proactive Data Entry Governance**
* **Objective:** Building automated spreadsheet constraints to block bad data input at the source, preventing human entry errors.
* **Core Techniques Used:** Enforcing strict text-only or numerical constraints, utilizing logical character lengths (e.g., exact 11-digit phone numbers), custom duplicate-blocking formulas, and deploying dynamic dropdown location lists.

### 📁 [03_Excel_Formulas](./03_Excel_Formulas/)
**Focus: Advanced Logic, Strings, & Calculation Engines**
* **Objective:** Implementing robust formulas to handle relational data extraction, advanced mathematical filters, and text mining.
* **Core Techniques Used:**
  * *Lookup Matrices:* `VLOOKUP`, `HLOOKUP`, and flexible `INDEX & MATCH` architectures.
  * *Text Parsing & Cleaning:* `SUBSTITUTE`, `LEFT`, `MID`, `RIGHT`, and `FIND`.
  * *Conditional Math & Logic:* `SUMIFS`, `COUNTIFS`, `AVERAGEIFS`, `SUBTOTAL`, `SUMPRODUCT`, and layered `NESTED IF` logical blocks.
