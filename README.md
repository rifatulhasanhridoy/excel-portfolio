# Excel Analytics Masterclass Portfolio

Welcome to my comprehensive Excel analytics portfolio! This repository serves as a centralized, structured database showcasing my progression through data cleaning, entry governance, advanced calculation design, and modern data transformation with Power Query.

---

## 🗂️ Portfolio Modules

### 📁 [01_Data_Cleaning](https://github.com/rifatulhasanhridoy/excel-portfolio/blob/main/01_Data_Cleaning)
**Focus: Structural Data Preprocessing & Pipeline Building**

- **Objective:** Transforming chaotic, multi-source tracking records into pristine, analysis-ready tabular models.
- **Core Techniques Used:** Strip hidden structural spacing errors (`TRIM`), fix inconsistent text casings (`PROPER`, `UPPER`), isolate fragmented data blocks (`Text to Columns`), and eliminate exact record redundancy.

---

### 📁 [02_Data_Validation](https://github.com/rifatulhasanhridoy/excel-portfolio/blob/main/02_Data_Validation)
**Focus: Proactive Data Entry Governance**

- **Objective:** Building automated spreadsheet constraints to block bad data input at the source, preventing human entry errors.
- **Core Techniques Used:** Enforcing strict text-only or numerical constraints, utilizing logical character lengths (e.g., exact 11-digit phone numbers), custom duplicate-blocking formulas, and deploying dynamic dropdown location lists.

---

### 📁 [03_Excel_Formulas](https://github.com/rifatulhasanhridoy/excel-portfolio/blob/main/03_Excel_Formulas)
**Focus: Advanced Logic, Strings, & Calculation Engines**

- **Objective:** Implementing robust formulas to handle relational data extraction, advanced mathematical filters, and text mining.
- **Core Techniques Used:**
  - **Lookup Matrices:** `VLOOKUP`, `HLOOKUP`, and flexible `INDEX & MATCH` architectures.
  - **Text Parsing & Cleaning:** `SUBSTITUTE`, `LEFT`, `MID`, `RIGHT`, and `FIND`.
  - **Conditional Math & Logic:** `SUMIFS`, `COUNTIFS`, `AVERAGEIFS`, `SUBTOTAL`, `SUMPRODUCT`, and layered `NESTED IF` logical blocks.

---

### 📁 [04_Power_Query](https://github.com/rifatulhasanhridoy/excel-portfolio/blob/main/04_Power_Query)
**Focus: Automated Data Import, Transformation & Consolidation**

- **Objective:** Leveraging Excel's Power Query engine to automate data ingestion from diverse sources, reshape raw datasets, and consolidate multi-file structures into unified, refresh-ready models — eliminating manual ETL effort.
- **Chapters:**

  | Chapter | Topic | Key Skills |
  |---------|-------|------------|
  | **Chapter 1** | Data Cleaning | Removing duplicates, fixing data types, trimming whitespace, handling nulls |
  | **Chapter 2** | Importing Data from Various Sources | Import from PDF, Web, multiple sheets; Pivot & Unpivot columns |
  | **Chapter 4** | Appending from Multiple Workbooks | Folder connector, combining annual files (2014–2018) automatically |
  | **Chapter 5** | Appending CSV Files | Merging regional CSVs (Central, North, South, West) into one dataset |
  | **Chapter 6** | Advanced Grouping | Multi-column Group By, custom aggregations, product & version data |

- **Core Techniques Used:** Power Query Editor, `Append Queries`, `Merge Queries`, folder-based dynamic data loads, Pivot/Unpivot transformations, data type enforcement, and step-by-step M query pipelines.
