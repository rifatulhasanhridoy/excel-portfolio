# 🏥 Hospital Emergency Room Dashboard

An interactive Excel dashboard built using Power Query, Pivot Tables, Pivot Charts, Slicers, and Excel formulas to analyze emergency room operations and patient trends.

---

## 📌 Project Overview

This dashboard provides an overview of emergency room performance, helping monitor patient flow, admissions, waiting time, satisfaction scores, and department referrals through interactive visualizations.

Users can filter the report by **Year** and **Month** to explore trends and compare key performance indicators.
